# 🎨 Design Reference — Maison De Son

## Fragrantica (Target UI)

Tài liệu tham khảo chính cho cấu trúc Authority-style.

### 🏠 Homepage
- `fragrantica_homepage_top`: Header, Search, Hero Article.
- `fragrantica_homepage_middle`: News Feed, Sidebar Latest Reviews.
- `fragrantica_homepage_bottom`: Content blocks.
- `fragrantica_homepage_footer`: Footer.

### 🧴 Product Page
- `fragrantica_product_top`: Brand, Name, Image, Main Accords bars.
- `fragrantica_product_ratings_desc`: Star rating breakdown, Pros/Cons, Description.
- `fragrantica_product_pyramid`: Icon-based pyramid (Top, Middle, Base notes).
- `fragrantica_product_related_news`: Carousel liên quan.

---

## Cách sử dụng

**Anti & Coder:**
1. Tham chiếu ảnh trong `docs/design-reference/fragrantica/` để xác định:
   - Layout grid (spacing, padding).
   - Hierarchy (font-size, color).
   - Interactive blocks (Main accords, Ratings).
2. Khi code component mới, bắt buộc phải đối chiếu với các screenshots này để đảm bảo độ chính xác theo mẫu.
