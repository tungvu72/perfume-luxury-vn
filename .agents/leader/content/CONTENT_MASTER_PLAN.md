## 🖼️ Image Workflow

### Process Overview

```
Research → Capture → Edit → Compress → Upload → Optimize
```

### Step-by-Step

#### 1. Research (15 min/image set)
```
□ Find official brand images
□ Check Fragrantica for community photos
□ Search YouTube for review thumbnails
□ Plan own photos if product in hand
```

#### 2. Capture (30 min/image set)
```
□ Take photos (bottle, box, details)
□ Screenshot from sources (if allowed)
□ Download from official sites
□ Create notes pyramid graphic
```

#### 3. Edit (20 min/image set)
```
□ Crop to standard sizes
□ Adjust brightness/contrast
□ Add subtle branding
□ Create comparison graphics
```

#### 4. Compress (10 min/image set)
```
□ Run through TinyPNG.com
□ Target: < 200KB per image
□ Check quality after compression
□ Convert to WebP format
```

#### 5. Upload (10 min/image set)
```
□ Upload to Sanity CMS
□ Add descriptive filenames
□ Write alt text (100-125 chars)
□ Set width/height attributes
```

#### 6. Optimize (5 min/image set)
```
□ Enable lazy loading
□ Add srcset for responsive
□ Verify in PageSpeed Insights
□ Fix any warnings
```

- [ ] Author bio với credentials
- [ ] Test duration stated (minimum 7 days)
- [ ] Original photos included
- [ ] Sources cited (2+ links)
- [ ] Facts verified (notes, price, availability)
- [ ] No exaggerated claims
- [ ] Disclosure (nếu affiliate/sponsored)
- [ ] Date published + last updated
- [ ] Internal links (3-5)

### Post-Publish

- [ ] Share on social media
- [ ] Respond to comments (48h)
- [ ] Monitor rankings (weekly)
- [ ] Update if info changes
