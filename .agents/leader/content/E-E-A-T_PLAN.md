# E-E-A-T COMPLIANCE PLAN

## 🎯 Overview

E-E-A-T là tiêu chuẩn của Google để đánh giá chất lượng content:
- **E**xperience (Trải nghiệm)
- **E**xpertise (Chuyên môn)
- **A**uthoritativeness (Thẩm quyền)
- **T**rustworthiness (Đáng tin)

Áp dụng cho YMYL (Your Money Your Life) niches - trong đó có product reviews & recommendations.

---

## 1. EXPERIENCE (Trải nghiệm)

### ✅ Yêu cầu tối thiểu

| Yếu tố | Standard | Proof |
|--------|----------|-------|
| Test sản phẩm | Tối thiểu 7 ngày | Nhật ký mùi hương |
| Ảnh thực tế | Không chỉ stock | Tự chụp bottle, wear tests |
| So sánh | Với 2+ products cùng category | Comparison tables |
| Long-term review | Update sau 30 ngày | Follow-up content |

## 4. TRUSTWORTHINESS (Đáng tin)

### ✅ Information Accuracy

| Check | Method |
|-------|--------|
| Notes accuracy | Verify 2+ sources |
| Price accuracy | Check 3+ retailers |
| Availability | Verify current stock |
| Claims | Substantiate with evidence |

### ✅ Transparency

```
## Disclosure Policy

- Some links are affiliate links (we earn commission)
- Products reviewed are purchased by us unless stated
- We do not accept payment for positive reviews
- Full transparency on sponsored content
```

### ✅ Site Requirements

- [ ] **About Us page** - Team, mission, story
- [ ] **Contact page** - Email, phone, address
- [ ] **Privacy Policy** - GDPR compliant
- [ ] **Terms of Service** - Clear policies
- [ ] **Return/Refund Policy** - If e-commerce
- [ ] **SSL Certificate** - HTTPS enabled
- [ ] **Fast loading** - Core Web Vitals pass

### ✅ No Misleading Claims

```
❌ TRÁNH:
- "Số 1 Việt Nam" (không bằng chứng)
- "Tốt nhất thế giới" (subjective)
- "100% chính hãng" (không thể guarantee)
- "Rẻ nhất thị trường" (price changes)

✅ THAY BẰNG:
- "Một trong những lựa chọn hàng đầu"
- "Được cộng đồng đánh giá cao"
- "Cam kết chính hãng, hoàn tiền nếu fake"
- "Giá cạnh tranh trong phân khúc"
```

### ✅ Review Integrity

```
## Our Review Process

1. Purchase product (or receive sample, disclosed)
2. Test minimum 7 days
3. Document experience daily
4. Compare with similar products
5. Write honest review (pros AND cons)
6. Update after 30 days if needed
7. Respond to reader comments
```

## 📋 E-E-A-T Checklist (Per Article)

### Before Publishing

- [ ] Author bio included with credentials
- [ ] Experience demonstrated (test duration, photos)
- [ ] Sources cited (minimum 2)
- [ ] Facts verified (notes, price, availability)
- [ ] No exaggerated claims
- [ ] Disclosure included (if affiliate/sponsored)
- [ ] Contact info accessible
- [ ] Date published + last updated
- [ ] Comments enabled (for engagement)
- [ ] Internal links to related content
- [ ] External links to authoritative sources

### After Publishing

- [ ] Monitor comments (respond within 48h)
- [ ] Track performance (rankings, traffic)
- [ ] Update if information changes
- [ ] Add user testimonials if received
- [ ] Share on social (engagement signals)
