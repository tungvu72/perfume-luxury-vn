# Fragrance YouTube System

Use this skill as an operating system, not a generic idea list.

## Core rule
Always start from channel evidence before suggesting new ideas.

Read these in order when they exist:
1. `references/audience-model.md`
2. `references/format-mix.md`
3. `references/workflow.md`
4. `references/files-to-maintain.md` when setting up or updating the memory loop

## What to optimize for
- Broad male fragrance demand
- Repeatable winner formats
- Low topic drift
- Fast production decisions
- Fewer repeated flops

## Channel logic this skill assumes
Default winning lanes:
- Ranking videos = growth engine
- Battle videos = comment/debate engine
- Problem-solving videos = trust + buyer intent
- Opinion/identity videos = loyalty builder

Do not overfill the calendar with one lane.

## Operating workflow
1. Identify whether the request is about planning, ideation, diagnosis, or postmortem.
2. Anchor recommendations to actual audience/analytics evidence.
3. Separate:
   - format problem
   - topic problem
   - packaging problem
4. Prefer specific next actions over broad brainstorming.
5. Log durable learnings after important strategy work.

## Guardrails
- Do not copy competitor scripts or angles verbatim.
- Extract patterns, not wording.
- Do not treat niche fragrance topics as default growth topics.
- Do not recommend women-focused content unless channel evidence supports it.
- Do not keep repeating ranking videos past the fatigue threshold; use the format mix guidance.

## Output preference
When giving plans, prefer:
- clear topic choice
- why it fits the audience
- where it sits in the format mix
- what should publish before/after it

## If building files or trackers
Use the following durable file types when useful:
- winner patterns
- flop graveyard
- topic backlog
- competitor outliers
- audience model
- performance log

See `references/files-to-maintain.md` for structure.
