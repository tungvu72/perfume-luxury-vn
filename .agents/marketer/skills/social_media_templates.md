# Skill: Social Media Templates

## INSTAGRAM — Caption nước hoa cao cấp

### Dạng 1: Cảm xúc / Storytelling
```
[Emoji mở đầu] [Câu mở gợi cảm xúc — không đề cập tên sản phẩm ngay]

[Đoạn mô tả hương thơm qua cảm nhận 2-3 câu]

✨ [Tên sản phẩm] — [tagline ngắn gọn]

[Thông tin thực tế: lưu hương, giá, link]

[3-5 hashtag tiếng Việt] [3-5 hashtag tiếng Anh]
```

### Dạng 2: Giáo dục (Edu-content)
```
🌿 Bạn có biết: [fact thú vị về hương thơm / thành phần]

[Giải thích ngắn 2-3 câu dễ hiểu]

👇 Trải nghiệm điều này tại [Tên sản phẩm]:
• [Điểm nổi bật 1]
• [Điểm nổi bật 2]
• [Điểm nổi bật 3]

[Link / CTA]
#nướchoa #perfume #[tên brand]
```

### Hashtag chuẩn cho tài khoản nước hoa VN:
```
Tiếng Việt:
#nướchoa #nuochoacaocap #nuochoachinhang #reviewmuahoa #nuochoahanoi #nuochoasaigon

Tiếng Anh:
#perfume #fragrancecommunity #perfumelover #scentoftheday #luxuryperfume #fragrancereview
```

```

[Body 3-5 câu — thông tin giá trị]

👉 [Sản phẩm của chúng tôi giải quyết điều này như thế nào]

✅ [Benefit 1]

✅ [Benefit 3]

🛒 Xem tại: [link]
💬 Bạn đang dùng nước hoa gì? Comment bên dưới nhé!
```

## TIKTOK — Script video 30-60 giây

### Hook (3 giây đầu — quan trọng nhất):
```
Option 1: "Nước hoa này bị người ta ghét vì quá [tính từ mạnh]..."
Option 2: "[Con số gây sốc] người đang dùng sai nước hoa theo cách này..."
Option 3: "POV: Bạn vừa ngửi được mùi hương đắt nhất thế giới..."
```

### Cấu trúc video 60s:
```
0-3s:   Hook (câu mở giật tít)
3-15s:  Problem / Bối cảnh  
15-45s: Solution / Demo sản phẩm / Thông tin giá trị
45-55s: CTA rõ ràng ("Follow để xem thêm review...")
55-60s: Outro ngắn
```

## EMAIL MARKETING — Newsletter nước hoa

### Subject line formula:
```
[Tên người] ơi, [trigger cảm xúc hoặc câu hỏi tò mò]
VD: "Hè này bạn đang thiếu mùi hương này..."
VD: "3 nước hoa dưới 2 triệu đang được săn đón nhất tháng 3"
```

### Cấu trúc email:
```
Greeting cá nhân hoá
↓
Hook / Câu chuyện ngắn (3-5 câu)
↓
Sản phẩm spotlight (ảnh + mô tả 50 từ + giá + CTA button)
↓
Tips nhỏ (1 mẹo dùng nước hoa)
↓
CTA chính + Link website
```
