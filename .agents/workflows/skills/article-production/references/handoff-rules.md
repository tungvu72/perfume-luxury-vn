# Handoff Rules

Nguồn gốc: rút gọn từ `ANTI_TASK_TEMPLATE.md` và workflow đã chốt với Anti.

## Coder làm gì
- viết content thuần
- lưu `.md` trong `content/drafts/`
- giữ paragraph spacing đúng markdown
- thêm image brief cuối bài
- không thêm frontmatter nếu không được yêu cầu

## Anti làm gì
- thêm frontmatter
- thêm internal links
- thêm ảnh / alt / placement
- render check
- deploy

## File handoff tốt phải có
- title/H2 structure rõ
- content final
- image brief
- nếu cần, note CTA direction

## Không làm
- không paste full article trên Telegram nếu có thể lưu file
- không giao việc chỉ bằng chat khi task quan trọng
- không để Anti phải đoán format paragraph

## Tin nhắn bàn giao ngắn
Khi đã có file, chỉ báo:
- file path
- loại bài
- đã có image brief hay chưa
